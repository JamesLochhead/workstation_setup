# Installation

1. Run the bootstrap scripts in `macos/monterey` to install Homebrew and the
   packages that Homebrew installs.

2. Insert OpenSSH private key.

3. Login to iCloud such that `workstation_setup` is available.

4. Run `git_config.sh` and then `setup_directory_structure.sh`.

5. Run the Firefox scripts in order.

6. Configure iterm2 to use the saved configuration and update it automatically.
